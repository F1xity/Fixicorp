Nom de la balise Ce qu'elle fait pour aller plus loin
< html > est une balise qui permet de coder dans l'environement HTML.
< head > est une balise où on entre le titre de la page et les paramètres principaux du la pages .
< title > est une balise permettant de mettre le texte defini en temps que titre. Exemple
< body > est une balise dans la quelle on rentre tout les textes et autres que nous voulons afficher sur notre page.
< h1,h2,h3... > est une balise qui permet d'afficher un texte defini avec une taille choisi. Exemple
< center > est une balise qui met le texte defini au centre de la page. Exemple
< br > est une balise qui permet de retourner a la migne sur le navigateur. Exemple
< u > est une balise qui souligne le texte defini. Exemple
< i > est une balise qui met en italique le texte defini.
< b > est une balise qui met le texte defini en gras.
< a href > est une balise qui permet de connecter 2 page web ensemble. Exemple
< img > est une balise qui permet d'incorporer une image dans notre page web. Exemple
< table > est une balise qui permet d'ouvrir un tableau. Exemple
< caption > est une balise qui donne le nom du tableau.
< tr > est une balise qui créé une ligne du tableau.
< td > est une balise qui sépare chaque élement de la ligne.
< th > est une balise qui donne le nom des colonnes.
< ol > est une balise qui permet de créé des listes numérotées. Exemple
< li > est une balise qui sépare les différent élements d'une liste.
< ul > est une balise identique a ol mais qui affiche des points à chaque élements de la liste
< sup > est une balise qui affiche le texte défini en exposant. Exemple
< sub > est une balise qui affiche le texte plus bas que la normal.
< mark > est une balise qui surligne le texte.
< form > est une balise qui permet de créer des formulaires. Exemple
< input > est une balise qui permet de rentrer de nombreuse données différentes.
< label > est une balise qui donne un nom à l'input associé.
< textarea > est une balise qui créé une zone de texte ajustable.
< fieldset > est une balise qui permet d'encadrer graphiquement des membres d'un formulaire.
< legend > est une balise qui permet de donner un nom à notre fieldset.
< select > est une balise qui créé un menu déroulant.
< option > est une balise qui sépare chaque choix du menu déroulant.
< optgroup > est une balise qui permet de grouper les différents choix d'un menu déroulant.
< p > est une balise qui permet délimite un paragraphe. Plus tard
< div > est une balise qui permet de grouper un ensemble d'élements.
< span > est une balise qui change les paramètres d'un texte défini.
